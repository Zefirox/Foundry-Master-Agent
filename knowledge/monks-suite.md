# Monk's Modules Suite — 11 Módulos de Utilidades

## Resumen

Colección de módulos de "TheRipper93" que añaden utilidades QoL a Foundry.

## Módulos

### Monk's Active Tile Triggers v13.06
Dispara eventos al pisar/entrar/salir de tiles.

```js
// Configuración via UI del tile
// Triggers disponibles:
// - Token enters tile
// - Token exits tile
// - Token moves within tile
// - Click on tile
// - Double-click on tile

// Acciones disponibles:
// - Execute macro
// - Move token
// - Toggle tile
// - Play sound
// - Show/hide drawing
// - Change scene
// - Roll table
```

### Monk's Bloodsplats v13.01
Sangre visual en tokens según HP.
- Automático: más daño = más sangre
- Configurable: tamaño, opacidad, estilo

### Monk's Combat Details v13.05
Info detallada de combate en chat cards.
- Damage breakdown
- Resistance/vulnerability info
- Critical hit details

### Monk's Combat Marker v12.01
Marca turnos y condiciones en el tracker de combate.

### Monk's Common Display v13.01
Muestra elementos comunes en todas las pantallas (ideal para mesa presencial).

### Monk's Enhanced Journal v13.06
Diario mejorado con:
- Links entre entradas
- Embeds de contenido
- Better formatting
- Scene links

### Monk's Little Details v13.03
QoL variado:
- Auto-target tokens
- Token resize
- Better drag-drop
- HUD improvements

### Monk's Sound Enhancements v13.02
Sonidos ambientales por escena:
- Background music per scene
- Ambient sounds
- Sound transitions

### Monk's TokenBar v13.02
Barra lateral de tokens para acceso rápido.

### Monk's Wall Enhancement v13.04
Walls avanzados:
- One-way walls
- Wall height (integración con wall-height)
- Better wall rendering

### Monk's Hotbar Expansion v13.01
Hotbar expandido con más slots y mejor organización.

## TheRipper93's Module Hub v5.0.7

- Configuración centralizada
