# Chris's Premades (Cauldron of Plentiful Resources) v1.5.27

## Concepto

Biblioteca de items/spells/features pre-configurados con toda la automatización
(DAE + MidiQOL + Active Effects + animaciones) ya lista para usar.

## Contenido

Compendios con contenido pre-construido:

| Compendio | Contenido |
|---|---|
| CPR Spells | Hechizos con DAE effects, MidiQOL macros, animaciones |
| CPR Class Features | Features de clase automatizados |
| CPR Feats | Feats con efectos dinámicos |
| CPR Items | Items mágicos con automatización |
| CPR Monster Features | Features de monsters |

## Uso

1. Abrir el compendio desde Foundry
2. Drag-drop el item al actor
3. La automatización se configura automáticamente

## Ventajas

- **No requiere configuración manual**: todo viene pre-configurado
- **Compatibilidad garantizada**: testeado con MidiQOL + DAE + Times Up
- **Animaciones incluidas**: mapeo JB2A ya configurado
- **Macros incluidas**: macros de MidiQOL embebidas en items

## Ejemplo de Item Pre-construido

Un "Fireball" de CPR incluye:
- Damage roll correcto (8d6 fire damage)
- DEX save con DC del caster
- Active Effect: "On fire" en targets que fallan
- MidiQOL macro: animación JB2A al castear
- Concentration: N/A (instantáneo)
- Scaling: automático a higher level (11d6 at 4th level, etc.)

## Dependencias

- Requiere: MidiQOL
- Requiere: DAE
- Requiere: Times Up
- Recomendado: Sequencer + JB2A
- Recomendado: Active Auras
